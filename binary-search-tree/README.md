# Двоичное дерево поиска

## Суть задачи

Нужно реализовать две базовые операции над BST:

- `insert(int val)` — вставка элемента
- `remove(int val)` — удаление элемента

Дерево используется как множество, поэтому дубликаты не добавляются.

## Реализация

Основные файлы:
- `tree/tree.h` — структура узла и дерева
- `tree/tree.cpp` — реализация `insert` и `remove`
- `tree/test.cpp` — тесты

## Как работает `insert`

Вставка реализована итеративно:

1. Если корень пустой, создаётся новый узел.
2. Иначе идём вниз по дереву:
   - если значение меньше текущего — идём влево
   - если больше — идём вправо
   - если равно — возвращаем существующий узел
3. Когда найдено пустое место, создаётся новый узел.

## Как работает `remove`

Удаление реализовано для всех трёх классических случаев:

1. Узел не найден — вернуть `false`.
2. У узла нет детей или только один ребёнок:
   - родительская ссылка переназначается на существующего ребёнка
3. У узла два ребёнка:
   - ищется минимальный элемент в правом поддереве
   - его значение копируется в удаляемый узел
   - затем узел-преемник удаляется как более простой случай

## Какие алгоритмы и подходы применены

- итеративный спуск по BST
- поиск inorder-преемника
- хранение детей через `std::unique_ptr`
- переназначение владения через `std::move`

## Сложность

В среднем:
- вставка: `O(h)`
- удаление: `O(h)`

где `h` — высота дерева.

## Как собрать и запустить

```bash
cmake -S . -B build
cmake --build build
ctest --test-dir build
```

## Что важно в реализации

- дерево не балансируется автоматически
- память управляется безопасно, без явных `new` и `delete`
- удаление корректно обрабатывает случаи 0, 1 и 2 потомков
