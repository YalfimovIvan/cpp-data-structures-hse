# Изограмма

## Суть задачи

Нужно определить, является ли строка изограммой, то есть содержит ли она только уникальные буквы.

Особенности постановки:
- регистр букв не важен
- небуквенные символы игнорируются
- повтор дефиса или пробела не делает строку невалидной

## Реализация

Основная функция:

```cpp
bool is_isogram(std::string const& word);
```

Код находится в `isogram/isogram.cpp`.

## Как работает алгоритм

В реализации используется `std::unordered_set<char>`.

Алгоритм:

1. Строка проходится посимвольно.
2. Для каждого символа проверяется, является ли он буквой через `std::isalpha`.
3. Буква приводится к верхнему регистру через `std::toupper`.
4. Если такая буква уже встречалась в `unordered_set`, функция возвращает `false`.
5. Иначе буква добавляется в множество.

Если повторов не найдено, возвращается `true`.

## Какие алгоритмы и структуры применены

- хеш-множество `std::unordered_set`
- нормализация регистра
- линейный проход по строке

## Сложность

- по времени: в среднем `O(N)`
- по памяти: `O(K)`, где `K` — число различных букв

## Как собрать и запустить

```bash
cmake -S . -B build
cmake --build build
ctest --test-dir build
```
