# Структуры данных и алгоритмы на C++


Важная особенность всех задач: решения написаны в стиле учебного modern C++, где активно используются:
- `std::unique_ptr`
- `CMake`
- модульные тесты
- разбиение логики по небольшим функциям, ООП

## Содержимое

- `isogram` — проверка строки на изограмму
- `linked-list-merge-sort` — сортировка слиянием для односвязного списка
- `binary-search-tree` — вставка и удаление в двоичном дереве поиска
- `bst-bidirectional-iterator` — двунаправленный итератор для BST

## Какие идеи отрабатываются

- использование хеш-структур для проверки уникальности
- merge sort на связных списках
- операции над деревом поиска
- обход BST в инфиксном порядке
- работа с указателями на родителей
- безопасное владение памятью

## Как собирать проекты

Каждая задача является отдельным CMake-проектом. Команды запускаются из нужной подпапки.

Пример:

```bash
cd isogram
cmake -S . -B build
cmake --build build
ctest --test-dir build
```

Аналогично для остальных подпроектов:

```bash
cd linked-list-merge-sort
cmake -S . -B build
cmake --build build
ctest --test-dir build
```
